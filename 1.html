Hypertext Markup Language (HTML) is the standard Markup language for creating web pages and web applications. Web browsers receive HTML documents grom a web server or from local storage and render tham into multimedea web pages. HTML describes the stucture of a web page semantically and originally included cues for the appearance of the document.

HTML elenents are the building blocks of HTML pages. With HTML constructs, images and other onjects, such as interactive foms, may be embedded into the rendered page. It provides a means to create structured documents by denoing structural semantics for text such as headings, paragraghs, lists, links, quotes and other items. HTML elements are delineated by tags, written using angle brackets.
